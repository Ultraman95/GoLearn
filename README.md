GoLearn
--------------------------------------------
        本项目的主旨是方便初学者练习动手.
        通过代码学习go语言，代码通过各种实例讲解go的基本使用方法，以及go中需要了解的知识点.
   

1.下载本项目  
2.配置go环境  
3.使用go module构建项目  
    &emsp;&emsp;a.初始化 go mod init goLearn ，这里的goLearn可以自己取名（代表生成应用程序的名称）  
    &emsp;&emsp;b.go build 构建生成可执行程序
    
项目标记解释   
(****)代表常用度，星号越多表示越常用
